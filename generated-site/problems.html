Question 3. If, for all xxx, f′(x)=(x−2)4(x−1)3f'(x) = (x-2)^4(x-1)^3f′(x)=(x−2)4(x−1)3, it follows that function fff has…

  • B. a relative maximum at x=1x = 1x=1
  • C. both a relative maximum at x=1x = 1x=1 and a relative maximum at x=2x = 2x=2
  • D. neither a relative maximum nor a relative minimum

We know that there may be extrema at points where f′(x)=0f'(x) = 0f′(x)=0, which are x=1x = 1x=1 and x=2x = 2x=2. We can determine whether these points are minima or maxima using the second derivative test.

f′′(x)=4(x−2)3(x−1)3+3(x−1)2(x−2)4f''(x) = 4(x-2)^3(x-1)^3 + 3(x-1)^2(x-2)^4f′′(x)=4(x−2)3(x−1)3+3(x−1)2(x−2)4

f′′(1)=0f''(1) = 0f′′(1)=0, so it's an inflection point and not an absolute minimum/maximum. The same goes for f′′(2)f''(2)f′′(2), so the answer is D.
